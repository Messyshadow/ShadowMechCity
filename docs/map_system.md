# 地图系统设计（阶段3 蓝图）

> 参考用户提供的「地图系统 MAP SYSTEM」设计图。
> 核心：探索未知区域，解锁地图，发现隐藏的秘密。获得新能力后返回旧区域会发现新路径和秘密（银河城环形探索）。

## 地图界面图例 (MAP INTERFACE)
| 图标 | 含义 |
|------|------|
| 玩家位置 | 当前角色所在 |
| 存档点 | 存档/恢复 |
| 传送点 | 快速传送 |
| 商店 (S) | 购买道具/武器 |
| NPC | 对话/任务 |
| 任务目标 | 当前任务指引 |
| Boss (骷髅) | Boss 房 |
| 隐藏区域 (点线) | 未发现的隐藏房 |
| 未探索区域 (灰实块) | 迷雾未探索 |
| 可破坏墙壁 (黄虚线) | 攻击/炸弹可破 |
| 单向门 | 仅单向通过 |
| 需要能力 (锁) | 需特定能力解锁 |

## 地图缩放 / 层级
- 缩放：LB/RB 或 +/-
- 地图分层：**上层 / 中层 / 下层**，Y 键切换（立体多层地图）

## 区域信息 (AREA INFO) 示例：废弃矿坑
- 探索度 68% · 完成度 4/8 · Boss 击败 1/2 · 宝箱收集 12/20 · 隐藏区域 3/6

## 操作 (CONTROLS)
L 移动地图 · R 缩放 · X 标记/取消标记 · Y 切换层级 · B 关闭地图

## 标记系统 (MARK SYSTEM)
- 最多 10 个自定义标记
- 标记类型：Boss / 位置 / 兴趣点 / 锁(能力门) / 宝箱 等

## 图例 (LEGEND)
- 实线=已探索；虚线=未探索
- 门：蓝=已开启 / 橙=未开启 / 紫=需要能力解锁
- 可破坏墙壁 · 隐藏区域入口 · 需要特定能力 · 单向通道

## 实现要点（阶段3）
- 房间网格 + 迷雾解锁；进入房间点亮对应格
- 区域着色（按区域不同色）
- 能力门（颜色区分开启状态/能力需求）
- 探索度/收集度统计
- 全屏地图 UI（M 键开关）+ 标记
