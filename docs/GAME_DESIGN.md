## 1. 游戏概述
- **类型**：2D 横版**银河恶魔城（Metroidvania）动作游戏**
- **基调**：**暗黑 + 机械**（"暗影机械城"），赛博/蒸汽朋克工业风
- **核心循环**：探索互联大地图 → 获得新移动能力 → 解锁旧区域隐藏路径（环形探索）→ 战斗/Boss → 成长（技能树/武器/道具）
- **视角操作**：可**左右来回自由探索**（非单向跑酷），上下多层
