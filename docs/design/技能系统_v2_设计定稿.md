# 技能系统 v2 · 设计定稿（阶段 9.1）

> 目标：把当前"纯被动数值树"升级为 **冒险岛/DNF 式 + 武器联动** 的主动技能体系。
> 本文为 9.1 定稿，9.2 实现框架 / 9.3 武器联动深化 / 9.4 特效+平衡。
> 决策依据：与用户+策划讨论（2026-06-27）。

## 一、三大方向（已拍板）

| 议题 | 选择 | 备注 |
|------|------|------|
| **技能触发** | **混合制（偏搓招）** | 搓招连招为主（方向+键），简单技/大招走热键。**考虑未来手柄版**：摇杆方向+面键搓招手感天然，热键留给大招。 |
| **武器联动** | **技能共享、随武器变形** | 同一套"技能原型"，每把武器有自己的变体表现/数值（剑气波↔震地波↔榴弹）。最能体现"武器联动"，切武器即换打法。 |
| **资源系统** | **技力(MP)**（默认，可调） | 平时缓回、命中敌人快速回蓝，鼓励贴脸输出。大招另用怒气槽。⚠️用户未最终拍板，如嫌复杂可一键改为"纯冷却CD"。 |

## 二、核心理念

- **一套输入，全武器通用**：玩家记住的是"招式输入"（如 ↓+技能=地面波），不必为每把武器重学键位。
- **武器决定招式形态**：同一输入在不同武器上是不同的技能（外形/范围/弹道/附加效果/数值），这就是"武器联动"。
- **保留现有被动树**：`skills_data.gd` 的被动树继续存在（生命/暴击/吸血/三段跳…），并新增"技力/技能强化"分支。主动技能是**叠加的新层**，不推翻旧系统。
- **职业流派 = 武器选择 × 被动加点**（自然涌现，不做硬性职业锁）：
  - 铁剑 → 高速连斩爆发流（暴击/吸血加点）
  - 重锤 → 霸体控场流（生命/范围加点）
  - 蒸汽炮 → 远程风筝流（技力/弹道加点）

## 三、技能原型表（输入 → 各武器变形）

> "技能原型(archetype)"按**输入**定义；每把武器填一份"变体"数据。新增武器只需补变体，不写新逻辑。

| 原型 | 输入(键盘) | 输入(手柄) | 铁剑 猎魂剑 | 重锤 破岩锤 | 蒸汽炮 |
|------|-----------|-----------|------------|------------|--------|
| **地面波** | ↓ + K | ↓ + ◯ | 剑气波(穿透·多段) | 震地冲击波(范围·小眩晕) | 滚地榴弹(落地爆炸AoE) |
| **上挑** | ↑ + K | ↑ + ◯ | 上挑斩(挑飞→接空连) | 上勾锤(高伤击飞) | 防空齐射(向上扇形弹) |
| **突进技** | → → + K | →→ + ◯ | 瞬步连斩(突进多段) | 蛮牛冲撞(霸体推进·击退) | 后跃齐射(拉距离+横扫弹) |
| **空中下砸** | 空中 ↓ + K | 空中↓ + ◯ | 流星斩(下落斩+落地波) | 天崩(大范围砸地震荡) | 俯冲轰炸(向下连射) |
| **环身爆发** | ↓ ↓ + K | 按住◯蓄力 | 旋风斩(已有·范围) | 震地圈(环形冲击) | 全向弹幕(360°喷射) |
| **大招(终结)** | 热键 V（满怒气） | 肩键 L1 | 千刃斩(多段聚怪斩) | 陨星重击(巨型砸地) | 过载炮击(蓄力贯穿巨炮) |

- K 键当前是"各武器重攻击"——并入本体系：**单按K=普通重攻击，方向+K=对应招式**。
- J 普攻三连段保留（轻攻击）；连段尾可自然接搓招技。
- 招式有**起手帧/收招帧**（可被打断、需走位），不做无脑循环。

## 四、资源与数值（默认方案）

- **技力(MP)**：上限默认 100；自然回复 ~6/s；命中敌人回 +8/次（贴脸更省蓝）。
- 每招耗蓝：地面波/上挑 15 · 突进/下砸 20 · 环身爆发 30。
- **怒气槽**：攻击/受击积累，满（100）可放**大招**，放完清空。
- 被动树新增分支「机械超频(青)」：技力上限+ / 技力回复+ / 技能伤害+ / 招式冷却-。
- 每招有**短独立冷却**（0.6~1.5s）防连刷；与技力双约束。

## 五、键位（含手柄前瞻）

| 功能 | 键盘 | 手柄(规划) |
|------|------|-----------|
| 轻攻击连段 | J | ✕/□ |
| 重攻击/方向技 | K（+方向搓招） | ◯（+左摇杆方向） |
| 大招 | V | L1 / R1 |
| 切武器 | Q | △ / 十字键 |
| 跳/冲刺/炸弹 | 空格 / Shift·L / F | A / R2 / L2 |

> 手柄版（阶段14.3）：搓招=左摇杆方向序列+◯，比键盘更顺；大招/技能用肩键。本设计**为手柄预留**，不锁死键盘。

## 六、数据结构草案（9.2 实现用）

```gdscript
# skills_active.gd —— 技能原型(按输入)
const ARCHETYPES := {
  "ground":  {"input":"down+skill",  "name":"地面波", "mp":15, "cd":0.8},
  "upper":   {"input":"up+skill",    "name":"上挑",   "mp":15, "cd":0.9},
  "dash_atk":{"input":"dash2+skill", "name":"突进技", "mp":20, "cd":1.0},
  "dive":    {"input":"air_down+skill","name":"下砸", "mp":20, "cd":0.7},
  "burst":   {"input":"downdown+skill","name":"环身爆发","mp":30,"cd":1.4},
  "ult":     {"input":"hotkey_v",    "name":"大招",   "rage":100},
}
# weapons.gd 每把武器新增 skills 字段: archetype_id -> {fx, projectile, dmg_mult, range, kb, sfx, anim}
#   例: 铁剑.skills.ground = {fx:"slash_wave", projectile:"sword_wave", dmg_mult:1.4, ...}
#       重锤.skills.ground = {fx:"quake",      shockwave:true,        dmg_mult:1.8, stun:0.4, ...}
```

实现要点：
- 输入识别：在 player.gd 加一个**搓招缓冲器**（记录最近方向输入+时序），匹配 ARCHETYPES。
- 释放：扣技力/查冷却 → 按当前武器的变体数据生成 hitbox/弹道/特效（复用现有 weapons/fx/projectile）。
- 大招：怒气满 → V/肩键触发当前武器终结技。
