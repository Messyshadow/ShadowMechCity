# 地图/区域设计指引（策划建议，做区域时遵循）

> 来源：策划反馈 + 手绘草图。做阶段10各区域时按此细化。

## 1. 中央车站 = 新手村（不刷小怪）
- 中央车站(Hub)是**安全区**：NPC 聚集地、商店、存档、传送网络中心。
- **不刷任何敌人**。✅ 已执行（移除 hub 敌人）。
- 后续在此放 NPC：铸造师/术士药剂师/地图解析师/遗物收藏家/赏金追踪者。

## 2. 单个场景要更大、更细、更复杂（不要太空旷粗糙）
当前每个房间太小、太"一条走廊"。每个场景应做成**更大 + 内部结构丰富**：
- **内部隔墙/平台迷宫**：像草图那样，房间内部有多段错落的墙体和平台，形成绕行/分层路径，而非一条平地。
- **多层立体**：上中下多层，平台高低错落，需要跳跃/冲刺/二段跳穿行。
- **多个出入口/传送门**：一个大场景可有**多个传送门**（草图里三个角落各一个），通向不同区域/房间。
- **死路与支线**：内部留小龛、隐藏房入口、宝箱位。
- 细节物件：管道、齿轮、灯光、背景层次，增加"不空旷"的观感。

## 3. 草图要点（参考）
- 一个宽大的封闭房间，四壁完整。
- 内部由若干横/竖墙段构成**迂回路径**（类似简易迷宫），玩家需绕行。
- **三个角落各一个传送门**（左下、右上、右下），中部还有断墙缺口形成通道。

## 4. 落实计划
- 本次：中央车站去敌（✅）。
- 阶段10 做各区域时：每个区域场景按"更大+内部迷宫+多层+多传送门+隐藏龛"标准重做/扩建。
- 传送门系统（多传送点互联）配合阶段3 已有的房间门系统扩展。
