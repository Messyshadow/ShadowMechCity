# 暗影机械城 — 游戏设计文档 (GDD)

> **项目代号**：Project Eclipse  
> **游戏类型**：2D 动作 · 银河恶魔城 · RPG 成长 · 平台跳跃  
> **参考作品**：暗影火炬城 / 空洞骑士 / 奥日与鬼火意志 / 血污：夜之仪式

---

## 世界观纲领

> 曾经辉煌的机械文明在一场「虚空浸蚀」中走向末路。  
> 腐败的暗影能量渗入每一个齿轮与蒸汽管道，将机器变为怪物，将亡者变为傀儡。  
> 玩家扮演唯一尚存意识的「机械猎魂者」，  
> 深入这座被黑暗侵蚀的机械巨城，寻找重启文明的核心——**光核（Lumina Core）**。

**核心美术关键词**：铸铁齿轮 · 虚空裂痕 · 蒸汽暗光 · 机械骸骨 · 幽蓝辉光

## 区域规划

---

### 一、中央枢纽站（Hub）

> **主题**：废弃的机械都市中心站，机械与人类文明最后的残影。  
> 锈迹斑斑的穹顶下，全息投影图仍在断续运转。

| 功能 | 说明 |
|------|------|
| 商店 | 购买强化材料、药剂、地图碎片 |
| 存档点 | 中央能量柱（复活锚点） |
| NPC | 铸造师、术士药剂师、地图解析师、遗物收藏家、赏金追踪者 |
| 传送网络 | 所有传送点汇聚于此 |

**初始宝箱**

- 生命碎片 ×1
- 齿轮币 ×100

### 二、废铁矿坑

> **主题**：地下矿道与熔岩腔穴，被虚空能量污染后，亡灵与机械残骸融为一体。  
> 遍布腐蚀性矿气，矿车仍在黑暗中自动滑行。

**解锁能力**：炸弹 `Bomb`（击败 Boss 后获得）

#### 普通敌人

| 名称 | 说明 |
|------|------|
| 矿魂机械傀儡 | 矿工亡灵被嵌入机械外壳，行动迟缓但攻击力高 |
| 齿轮翼蝠 | 背部嵌有旋转齿轮，俯冲时可切割玩家 |
| 钻头蠕虫 | 头部装有钻头，从地面突袭 |
| 熔铁史莱姆 | 由废弃熔铁与暗影能量构成，接触灼伤 |

#### 精英敌人

| 名称 | 说明 |
|------|------|
| 重甲铸魂骑士 | 穿戴厚重矿坑铠甲，暗影能量护盾可短暂无敌 |
| 爆破机械专家 | 投掷腐蚀炸弹，引爆后留下持续伤害区域 |

#### 陷阱

- 自动落石装置
- 熔铁喷管
- 失控矿车轨道
- 机械尖刺阵

**Boss**：`岩核机甲巨兽` — 身躯由矿石与机械骨架熔合，核心暴露时为弱点，会召唤小型钻头蠕虫协助

**隐藏区域**：6 处

### 三、蒸汽铸造厂

> **主题**：蒸汽朋克风格的大型工业厂区，管道纵横，传送带永不停歇。  
> 失控的自动化程序让机械工人永远在黑暗中"劳作"。

**解锁能力**：磁力钩爪 `Mag Grapple`（击败 Boss 后获得，可钩取金属墙面快速位移）

#### 普通敌人

| 名称 | 说明 |
|------|------|
| 暗影蒸汽机械兵 | 胸口锅炉注入虚空能量，会释放暗影蒸汽云 |
| 机械幽灵蜘蛛 | 半透明的机械蜘蛛，能穿透薄墙移动 |
| 虚空追踪无人机 | 锁定玩家后高速冲刺，可被炸弹击落 |

#### 精英敌人

| 名称 | 说明 |
|------|------|
| 重型暗影机甲 | 双肩装有蒸汽炮台，可切换近战 / 远程模式 |
| 暗能激光炮塔 | 固定式但旋转扫射，被虚空能量强化后激光呈紫色 |

#### 陷阱

- 高压蒸汽喷射柱
- 虚空感应齿轮
- 腐蚀激光网格
- 高压电流弧

**Boss**：`虚影蒸汽机甲 MK-II` — 双足机甲，核心注入暗影能量后可进入虚空形态（半透明，无法被普通攻击命中，需引爆炸弹触发实体化）

**隐藏区域**：7 处

### 四、腐化水道

> **主题**：城市地下排水系统，遭虚空污液侵蚀后，生物变异与机械残骸共生。  
> 腐化之光在黑暗水面折射出诡异的绿光。

**解锁能力**：水下推进器 `Aqua Thruster`

#### 普通敌人

| 名称 | 说明 |
|------|------|
| 机械变异鱼人 | 下半身为鱼形机械推进器，腐化液为弹幕攻击 |
| 虚空毒蛙机甲 | 背部携带毒液储罐，跳跃时释放腐蚀毒雾 |
| 暗影水蛇导管 | 形如巨型管道蛇，藏于管道内突然咬合 |

#### 陷阱

- 腐化液池
- 虚空暗流（强制移动）
- 机械尖刺阵
- 腐蚀气泡弹幕区

**Boss**：`深渊机械巨鳄` — 半机械化的远古巨鳄，尾部为金属推进桨，可控制水流形成旋涡；背部装甲需炸弹破除后才能攻击弱点

**隐藏区域**：5 处

### 五、遗迹神殿

> **主题**：文明前期遗留的神殿遗址，精密机关与古老超自然力量融合。  
> 石像与机械骨架并存，祭司的灵魂被封印于机械容器中延续仪式。

**解锁能力**：遗迹符文护盾 `Rune Shield`（击败 Boss 后获得，短暂无敌并反弹投射物）

#### 普通敌人

| 名称 | 说明 |
|------|------|
| 机械石像鬼 | 石质外壳内嵌机械骨架，受到冲击时裂开并释放暗影碎片 |
| 遗迹守卫机甲 | 持古代仪式长矛，护盾由遗迹符文强化 |
| 祭司机械容器 | 被封印的祭司灵魂操控机械容器漂浮攻击，以幽灵形态暂时逃脱容器 |

#### 机关

- 古代符文压力板（踩踏触发机关序列）
- 移动平台（由古老机械驱动）
- 激光符文阵列（需贴墙跳越过高台）
- 旋转刃轮机关

**Boss**：`远古机械守护神` — 四臂巨型机械战神，全身刻有遗迹符文，会召唤浮动符文盾；在低血量时符文激活，进入超自然狂暴形态

**隐藏区域**：8 处

### 六、虚空要塞

> **主题**：漂浮于城市废墟上方的机械战舰群，被虚空能量贯穿后内外扭曲。  
> 空间在此处不稳定，部分区域会发生短暂的虚空折叠。

**解锁能力**：暗影滑翔翼 `Shadow Glider`

#### 普通敌人

| 名称 | 说明 |
|------|------|
| 虚空裂翼飞龙 | 机械骨架构成的飞龙，翅膀由虚空裂痕填充，可短程瞬移 |
| 暗影机械鹰 | 高速俯冲，羽毛为金属刀片 |
| 虚空风暴法师机甲 | 穿着机械法袍，操控虚空风暴弹幕，被击中一定次数后切换法术类型 |

#### 陷阱

- 虚空强风气流区（需暗影滑翔翼驾驭）
- 空间扭曲雷击点
- 坠落空间裂缝
- 虚空炮台自动防御

**Boss**：`虚空天龙机甲` — 将巨型飞龙骸骨改造为战斗载具，骑手为虚空能量实体；两个阶段：第一阶段空中弹幕战，第二阶段坠地后近战机甲模式

**隐藏区域**：8 处

### 七、暗影王城（终章）

> **主题**：整座机械城的核心，虚空能量最浓郁之处。

**特点**

- 全游戏最高难度
- 全程精英敌人
- 连续 Boss 战
- 终局剧情解锁

## ARPG 等级与成长系统

### 角色等级系统

#### 基础规则

- 等级上限：**Lv.50**
- 击败敌人、Boss、完成任务均可获得**经验值（EXP）**
- 每次升级获得 **2 个技能点（SP）**，可自由分配到技能树
- 升级同时自动提升基础属性

#### 角色基础属性

| 属性 | 初始值 | 每级成长 | 说明 |
|------|--------|----------|------|
| 最大生命（HP） | 100 | +8 | 受到伤害的承载上限 |
| 最大法力（MP） | 50 | +3 | 释放技能消耗 |
| 攻击力（ATK） | 10 | +2 | 影响普攻与技能基础伤害 |
| 防御力（DEF） | 5 | +1 | 减少受到的物理伤害 |
| 暴击率（CRIT） | 5% | — | 通过技能点/装备提升 |
| 移动速度（SPD） | 200 | — | 通过技能点提升 |

#### EXP 成长曲线

```
升级所需 EXP = 基础值 × 等级^指数
基础值 = 100，指数 = 1.8

例：
  Lv.1→2   : 100 EXP
  Lv.5→6   : 约 1,500 EXP
  Lv.10→11 : 约 6,300 EXP
  Lv.20→21 : 约 27,000 EXP
  Lv.49→50 : 约 240,000 EXP
```

### 怪物等级系统

#### 等级分布规则

每个区域的怪物有固定的**等级范围**，玩家进入时可见敌人等级标识。

| 区域 | 普通怪等级 | 精英怪等级 | Boss 等级 |
|------|-----------|-----------|----------|
| 中央枢纽站 | — | — | — |
| 废铁矿坑 | Lv.1～8 | Lv.6～10 | Lv.10 |
| 蒸汽铸造厂 | Lv.8～15 | Lv.13～18 | Lv.18 |
| 腐化水道 | Lv.15～22 | Lv.20～25 | Lv.25 |
| 遗迹神殿 | Lv.22～30 | Lv.28～33 | Lv.33 |
| 虚空要塞 | Lv.30～38 | Lv.36～42 | Lv.42 |
| 暗影王城 | Lv.40～48 | Lv.46～50 | Lv.50 |

#### 怪物属性公式

```
怪物 HP   = 基础HP × (1 + 怪物等级 × 0.12)
怪物 ATK  = 基础ATK × (1 + 怪物等级 × 0.10)
怪物 DEF  = 基础DEF × (1 + 怪物等级 × 0.08)
怪物 EXP  = 基础EXP × 怪物等级 × 1.5
精英怪    = 普通怪属性 × 3，EXP × 5
Boss      = 普通怪属性 × 15，EXP × 20
```

#### 等级差伤害修正

```
玩家攻击怪物：
  等级差 ≤ 5  : 正常伤害（×1.0）
  高于玩家 6+  : 玩家伤害 ×0.8（弱点状态）
  低于玩家 6+  : 玩家伤害 ×1.2，EXP 减半

怪物攻击玩家：
  高于玩家 6+  : 受到伤害 ×1.3（压制状态）
  低于玩家 6+  : 受到伤害 ×0.8
```

#### 怪物强化词缀（精英怪专属）

精英怪随机附带 1～2 个强化词缀：

| 词缀名 | 效果 |
|--------|------|
| 虚空护盾 | 首次受到致命伤时触发一次护盾（吸收 30% 最大 HP）|
| 狂暴核心 | HP 低于 30% 时 ATK +50%，移速 +30% |

| 🔧 **虚空蒸汽局（VoidSteam Co.）** | 远程 · 控场 | 虚空蒸汽炮、磁力钩爪 | 蒸汽铸造厂 |

每件装备有 **5 个品质层级**，品质越高基础属性越强，升级费用越贵：

| 品质 | 颜色标识 | 特性 |
|------|---------|------|
| 标准版（Standard） | 灰色 | 基础属性，无附加效果 |
| 改良版（Enhanced） | 绿色 | +1 词缀槽 |

### 升级补丁机制（Patch Upgrade）

每件装备有 **5 个升级阶段**（+1 ～ +5），每阶段消耗**齿轮币**：

```
升级费用 = 基础费用 × 品质系数 × 升级阶段系数

基础费用（标准版）：
  +1 : 200 齿轮币
  +2 : 500 齿轮币
  +3 : 1,200 齿轮币
  +4 : 3,000 齿轮币
  +5 : 8,000 齿轮币

品质系数：
  标准版 ×1.0 | 改良版 ×1.5 | 精英版 ×2.2
  传奇版 ×3.5 | 虚空版 ×6.0
```

每次升级提升：

- **武器**：ATK +8%（每阶）
- **防具**：DEF +8%，HP +5%（每阶）
- **饰品**：根据饰品类型提升对应属性 +6%

### 个性化订阅（Cosmetic & Bonus Patch）

装备达到 +3 后可选购品牌**个性化补丁包**（不影响强度，纯外观 + 小额数值）：

| 补丁类型 | 内容 | 费用 |

| 音效补丁 | 更换攻击/受击音效 | 300 齿轮币 |
| 属性微调包 | 在该武器基础上小幅调整一项属性（±5%） | 800 齿轮币 |

## 黑客入侵系统（Hack System）

> 机械猎魂者的武器库中不只有钢铁，还有代码。  
> 通过随身携带的**量子渗透终端**，可对敌方机器人发动网络入侵，  
> 腐化其核心程序，让铁与钢变成你的爪牙。

### 解锁方式

完成蒸汽铸造厂支线任务「破解者遗产」后解锁黑客技能树；  
黑客系统通过 SP 独立升级，**等级越高入侵成功率越强、效果越多样**。

### 入侵成功率公式

```
基础成功率   = 黑客技能等级 × 15%（上限 90%）
等级差修正   = (玩家等级 - 敌人等级) × 5%
HP 状态修正  = 目标 HP 低于 50%：+10%；低于 25%：+25%
最终成功率   = min(基础成功率 + 等级差修正 + HP修正, 95%)

注：当敌人等级高于玩家 10 级以上时，
    入侵失败不仅无效，还会反向触发「信号追踪」——
    对玩家造成 ATK×30%～80% 的电磁反噬伤害。
```

### 黑客技能列表

| 技能名 | 效果 | 持续时间 | MP 消耗 | 冷却 |
|--------|------|---------|---------|------|
| 核心腐化 | 对目标机器人造成持续伤害（每秒 ATK×15%） | 8s | 20 MP | 10s |
| 临时征用 | 将目标机器人转为友军，但**不跟随玩家**，在原地攻击附近敌人 | 12s | 40 MP | 20s |
| 混乱指令 | 目标机器人进入混乱状态，随机攻击周围单位（含友军） | 6s | 30 MP | 15s |
| 系统过载 | 强制使目标机器人核心过热，3s 后爆炸（伤害：目标剩余 HP×40%） | 3s（引爆延迟）| 50 MP | 25s |
| 数据窃取 | 入侵成功后获取目标机器人数据，转化为少量 EXP 和材料 | 即时 | 15 MP | 8s |

> **Boss 机器人的入侵规则**：仅「核心腐化」和「数据窃取」对 Boss 生效，  
> 「临时征用」「混乱指令」「系统过载」均无效。

## 虚拟量子空间系统（Quantum Space System）

### 空间结构总览

### 二、战斗空间（Combat Space）

**开启条件**：战斗中，消耗 40 MP，按下量子战斗键  
**冷却时间**：45s（可通过技能点缩短）  
**效果**：在玩家周围展开一个**量子泡**，将范围内敌人短暂拖入折叠空间

#### 战斗空间持续时间

战斗空间的持续时间受**敌人等级和强度**影响：

| 目标强度 | 持续时间 |
|---------|---------|
| 普通敌人 | 15s |
| 精英敌人 | 10s |
| Boss（低于自身等级） | 6s |
| Boss（同级或更高） | 3s |
| 等级高出玩家 10+ | 1s（几乎无效，慎用）|

#### 战斗空间环境参数（加点解锁）

玩家可将 SP 投入「量子空间系」技能树，解锁可在战斗空间中部署的**环境设置**：

```
【量子空间系】
    ├── 空间稳定强化（最多 3 级）
    │       每级：战斗空间持续时间 +3s，冷却 -5s
    │       消耗：2 SP / 级
    │
    ├── 重力压缩场（最多 2 级）
    │       战斗空间内敌人移动速度 -30%，跳跃高度 -40%
    │       Lv.2：效果翻倍，玩家不受影响
    │       消耗：2 SP / 级
    │       前置：空间稳定 Lv.1
    │
    ├── 电磁干扰层（最多 2 级）
    │       战斗空间内敌人攻击间隔 +50%（类似攻速降低），
    │       弹幕类敌人弹幕速度 -30%
    │       消耗：2 SP / 级
    │       前置：空间稳定 Lv.1
    │
    ├── 量子镜像（最多 1 级）
    │       战斗空间内玩家产生 1 个镜像残影，
    │       分担 20% 敌方攻击伤害（残影有独立 HP，耗尽消失）
    │       消耗：4 SP
    │       前置：重力压缩场 + 电磁干扰层各 Lv.1
    │
    └── 扩张边界（最多 2 级）
            每级：战斗空间覆盖范围 +30%（影响收录判定范围）
            消耗：2 SP / 级
```

#### 机器人收录系统（Capture）

在战斗空间激活状态下，可尝试将**敌方机器人**收录进活动空间，供日后召唤：

```
收录成功率 = 基础成功率 + 血量加成 + 等级差修正

基础成功率：
  普通机器人     : 60%
  精英机器人     : 30%
  Boss 机器人    : 10%

血量加成（目标当前 HP / 最大 HP）：
  HP > 75%   : -20%（血多难收）
  HP 50%~75% : ±0%
  HP 25%~50% : +15%
  HP < 25%   : +35%（奄奄一息最好收）

等级差修正：
  玩家高出 5+  : +10%
  等级相近      : ±0%
  敌人高出 5+  : -15%
  敌人高出 10+ : -35%
```

**收录限制**：
- 同一种类的机器人只能持有 **1 台**（收第 2 台时需先释放或替换）
- Boss 机器人收录后不可再次出现于该区域，属稀有资源
- 收录失败不消耗道具，但战斗空间持续时间正常流逝

### 技术要求

- **引擎**：Godot 4.x
- **架构**：状态机（State Machine）驱动 PlayerController / EnemyAI / BossAI / RobotAI
- **关卡**：TileMap 系统，支持多层级（背景 / 碰撞 / 前景 / 暗影特效层）
- **存档**：支持多存档槽，持久化：探索进度 + 技能点分配 + 怪物击杀 + 机器人阵容 + 装备补丁状态
- **扩展性**：预留 DLC 区域接口，新区域可插件加载

### 代码生成原则

1. 所有系统生成完整可运行的 Godot 项目骨架
2. **LevelSystem**：`EXP = base × level^1.8`，升级广播信号触发属性更新
3. **SkillPointTree**：数据驱动（Resource），支持前置判断与运行时解锁
4. **StatSystem**：`基础值 + 等级成长 + 技能加成 + 装备加成`，等级差修正为最终乘数
5. **EnemyAI**：怪物等级由区域资源文件随机生成，精英词缀热插拔
6. **BossAI**：血量阈值触发多阶段相变
7. **SummonSystem**：维护上场机器人槽位（最多 3 个），召唤/收回/损毁均广播信号；RobotAI 使用独立状态机（待命/战斗/跟随/超载/损毁）
8. **HardwarePatchSystem**：装备对象携带品牌 ID + 品质等级 + 当前补丁等级，升级费用由公式动态计算；套装加成通过遍历已装备列表判断品牌数量触发
9. **HackSystem**：入侵成功率实时计算（黑客等级 + 等级差 + 目标 HP 比），失败反噬作为单独伤害事件广播；临时征用使目标切换阵营标签
10. **QuantumSpaceSystem**：活动空间以独立 Scene 加载（暂停主世界），战斗空间以 Area2D 泡区覆盖生效范围；CaptureSystem 按公式计算收录率并写入机器人数据库
