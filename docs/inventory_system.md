## 右侧：物品详情 (Item Detail)
示例：**影牙太刀 +5**（稀有·武器·已装备）
- 攻击力：物理攻击 256 · 攻击速度 1.25 · 暴击率 18%
- 附加属性：暴击伤害 +25% · 对人型敌人伤害 +12% · 精力恢复速度 +8%
- 武器技能：**影袭斩** — 快速突进斩击，对前方敌人造成伤害
- 风味文本："以黑影之铁打造，刀身轻盈而锋利，传说中属于某位隐于暗影的忍者。"
- 操作：卸下 · `V` 查看详情

## 物品稀有度
普通 / 稀有 / 史诗 / 传奇（颜色区分），对应宝箱掉落比例（见 GAME_DESIGN）。

## 实现要点
- 物品数据：id/名称/类型/稀有度/强化/属性/武器技能/风味/堆叠/重量/图标
- 装备影响派生属性（攻防暴击等）；每件武器自带一个武器技能
- 强化系统（+N 提升属性，消耗强化石）
- 货币：金币（普通）/紫晶（高级）
- UI：装备槽 + 网格 + 详情三栏；分类筛选/整理/丢弃
- 与技能树、地图、任务、档案共享统一主菜单（Q/E 切页）
