# 暗影机械城 — 分阶段开发计划（路线底稿）

> 目标：做成**商业成品**（以 `docs/design/暗影机械城_GDD(2).md` 策划权威方案为北极星）。
> 现实：完整 GDD（7区域/300+房间/50+敌/30+武器/10+Boss/20~40h）是团队+数月的体量，
> 无法一次做完。本计划把它拆成**一次一个明确可玩交付**的小阶段，token 花在刀刃上，可随时调整。

## 核心原则
1. **数据驱动**：加区域/敌人/武器/装备尽量只填数据（rooms/weapons/items_data/enemy_defs），少写新代码 → 单位成本低。
2. **每阶段必交付**：跑通无报错 → 截图验证 → 更新发布包（`E:\Godot\release\暗影机械城`）。
3. **美术/音频**：优先免费素材（Kenney/itch/rvros/OpenGameArt + 本地素材库）下载或程序生成、再修改统一风格。
4. **先系统后体量**：系统/框架做扎实后，内容靠批量扩充。
