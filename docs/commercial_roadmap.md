# 商业发布路线图 / 差距清单（持续更新）

> 记录距离"可上架商业游戏"还需补齐的维度，按优先级排列。每完成一项打勾。

## 一、商业发布差距维度（评估于阶段3完成时）

| 维度 | 现状 | 商业标准 | 优先级 |
|------|------|----------|--------|
| **存档系统** | ❌ 无 | 存/读进度、自动存档 | 🔴 必备 |
| **游戏外壳** | ❌ 无 | 开始菜单·暂停菜单·设置(音量/分辨率/改键)·手柄支持 | 🔴 必备 |
| **真·Boss战** | ⚠️ 仅强怪 | 多阶段·招式预警·血条·狂化·战斗机制 | 🟠 核心卖点 |
| **美术统一** | ⚠️ 混搭画风 | 统一专业画风(角色/敌人/场景/特效/UI) | 🟠 最影响卖相 |
| **音乐音效** | ⚠️ 借用/合成 | 原创分区域配乐·完整音效·环境音 | 🟠 |
| **内容量** | ⚠️ 5房间/9怪/3武器 | 数十~上百房间·多区域·多Boss·秘密收集 | 🟠 |
| **装备系统** | ❌ 无 | 装备槽/稀有度/强化/词条(见 inventory_system.md) | 🟡 |
| **叙事/世界观** | ❌ 无 | 剧情·NPC对话·碑文·开场/结局·引导教学 | 🟡 |
| **UI/UX 打磨** | ⚠️ 基础 | 转场动画·伤害数字·菜单美术·本地化 | 🟡 |
| **技术发布** | ⚠️ DEBUG包 | release导出模板·图标·优化·成就·Steam接入 | 🟡 |
| **QA/平衡** | ❌ | 数值平衡·难度曲线·playtest·bug修复 | 🟡 |

## 二、阶段进度
- ✅ 阶段1 战斗与武器系统（连击/3武器/可见挥砍/暗黑机械美术）
- ✅ 阶段2 技能与成长（技能树/经验/金币/道具拾取）
- ✅ 阶段3 银河城世界（5互联区域/四向门/钥匙门/存档点/地图M/分区域背景）
- ✅ 阶段4 游戏外壳 + 存档（开始菜单/暂停/设置/存读档/菜单键盘导航）+ 战斗重构（K=各武器不同重攻击 / 蒸汽炮真远程 / 下砸独立特效音效）
- 🔄 阶段5 **真·多阶段 Boss 战**（← 下一阶段；先升级主角攻击动画提升打击感）← 当前
- ⬜ 阶段6 装备/背包系统（见 inventory_system.md）
- ⬜ 阶段7 美术统一升级 + 原创音乐
- ⬜ 阶段7 **特色系统**（见 future_systems.md）：机器人召唤 / 虚拟量子空间 / 黑客入侵 / 装备升级
- ⬜ 阶段8 内容扩充（更多区域/敌人/Boss/秘密）
- ⬜ 阶段9 叙事/世界观/教学引导
- ⬜ 阶段10 技术发布（release导出模板去DEBUG/图标/优化/手柄/成就/上架）

> 特色玩法系统详见 [future_systems.md](future_systems.md)：4 个互相关联的机械朋克核心系统。

## 三、备注
- release 去 DEBUG 需编译 `scons target=template_release` 导出模板（约30-40分钟），再做标准导出。
- 美术统一通常需美术资源/外包；可考虑统一用某 CC0 像素风资源包替换全部素材。
