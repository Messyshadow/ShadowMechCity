# 10.1 蒸汽铸造厂 — 设计反馈（创意/QA 线程 · 给开发线程迭代）

> 性质：**设计质量反馈**（非报错）。10.1 功能/结构清单已达标且无报错（见 10.1-review）。本文是把它从"达标"推向"好玩好看"的具体改法。
> 实拍依据：`screenshots/qa6_factory_*.png`（四房宽景）。已同步更新 `docs/design/区域设计模板.md` 第二·五节"设计质量红线"。

## 通病（四房通用，按优先级）

### P1 · 底部"连续平地高速路"——最伤
四房最下方都是一整条平地，玩家贴底走基本就能横穿，迷宫/平台沦为"可选上层"。
- **改**：在底部开**坑/缺口/熔铁河**，把平台跳跃变成**必经路径**。临界路径至少有几段"必须跳/冲/爬"。

### P2 · 上半空旷、平台是孤立横条
大黑盒里几条漂浮横条，上 2/3 空。
- **改**：平台**连成可攀爬动线**（接二段跳/攀墙/滑翔）；把奖励/碎片放高处逼玩家上行；用内部竖墙真正切出**绕行回路**。

### P3 · 四房视觉雷同、无地标
熔炉大厅/装配车间/熔铁回廊/传送链区长得几乎一样。
- **改**：每房一个**招牌**（见下分房建议），一眼能认出"这是哪"。

### P4 · 陷阱是平地点缀
齿轮/陷阱摆在行走平地上 = 随机掉血。
- **改**：陷阱放在**缺口上方/窄道/必经跳点**，做成"看时机过"的挑战。（蒸汽阀建议改**周期喷发**+预警，比常驻判定更像机关——10.1 报告也提到了。）

## 分房招牌建议（兑现名字 = P5）

| 房间 | 招牌机制/地标（兑现名字） |
|---|---|
| 熔炉大厅 | 中央一座**巨型高炉**地标 + 上升**热气流**（踩上去顶玩家上跳，做垂直动线）；炉下熔铁池=底部危害 |
| 装配车间 | **传送带**（横向推动玩家/敌人，改变站位与节奏）+ 流水线上的**半成品机兵**（叙事彩蛋：敌人在这被造出来） |
| 熔铁回廊 | **熔铁河**贯穿底部，窄岩桥/断桥跨河，掉河即危害——直接兑现"熔铁"二字 |
| 传送链区 | **链条移动平台/升降机**（垂直/水平移动平台）做穿行，兑现"传送链" |

> 这些机制大多能复用现有 `hazard.gd`（周期喷发）、`oneways`/`platforms`（移动平台需小扩展：给平台加 `move` 字段做往返/升降）。传送带=对站在其上的 body 加横向速度。

## 我（创意线程）已顺手做的
- **升级了铸造厂背景**（`assets/bg/factory/*`，PIL 本地生成）：高炉穹顶+烟囱剪影、发光窗格、熔炉暖光辉光、近景大管道+齿轮——缓解 P3 视觉空旷/雷同的"背景那半"。布局那半仍需上面的改动。

## 边界
- 以上是**设计/手感**建议，最终好不好玩**需人工试玩**定（平台间距、热气流强度、传送带速度、熔铁河宽度）。
- 移动平台/传送带是新机制，做时注意：移动碰撞体若涉及 monitoring/shape 变更，沿用 `set_deferred`/`call_deferred`（见 9.2/9.3/过门 bug 教训）。
