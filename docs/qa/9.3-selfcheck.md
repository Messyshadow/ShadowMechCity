# 9.3 开发线程自查 — 技能×武器联动（供 QA 复验）

> 开发线程出具 · 日期 2026-06-27 · 范围：重锤/蒸汽炮全原型变体 + 环身爆发(新) + 下砸变形 + 「机械超频」被动。
> 参照 `docs/design/技能系统_v2_设计定稿.md` 第三节"输入→各武器变形"表。

## 各武器技能变形（同一输入，三套形态 — 便于画面验收）

| 原型 / 输入 | 铁剑(青) | 重锤(橙) | 蒸汽炮(紫) |
|---|---|---|---|
| **地面波** K | 剑气波(穿透飞弹) | 震地冲击波(贴地推进) | 远程能量弹(直线) |
| **上挑** 上+K | 小跳+青斩挑飞 | 大跳+橙冲击强击飞(霸体感) | **不跳**·向上扇形3弹(防空) |
| **突进** →→+K | 瞬步连斩(640速·残影) | 蛮牛冲撞(霸体iframes·强击退) | **后跃**+向前扇形3弹(拉距离) |
| **环身爆发** ↓↓+K (新) | 旋风斩(环身5道斩) | 震地圈(爆炸+环形强击退) | **全向弹幕**(360°×8弹) |
| **下砸** 空中↓+J | 流星斩(橙红双冲击) | 天崩(220半径+爆炸) | 俯冲轰炸(落点爆炸+左右扫射) |
| **大招** V(满怒气) | 千刃斩(3剑气+爆发) | 陨星重击(260爆炸·300半径) | 过载炮击(双发3x贯穿巨弹) |

形态差异手段：颜色取 `weapon.color`(青/橙/紫)；铁剑用斩击帧、重锤用 bolt+爆炸、蒸汽炮用 bolt2 飞弹/全向。

## 新增系统
- **环身爆发**：新原型，双击↓ arm 0.32s 窗口 + 仍按住↓ 才触发（`_try_skill` 优先级：↑上挑 > ↓↓爆发 > →→突进 > 地面波）。容错同突进，不与移动冲突。
- **机械超频被动**(`skills_data.gd`，青)：技力强化(上限+20×3) / 技力回流(回复+3/s×3) / 过载输出(技能伤害+15%×3) / 招式精通(冷却-15%×2)。已接入 `max_mp()` / 回复 / `_skill_dmg` / `_cast` 冷却。
- **projectile 任意角度**：新增 `vel:Vector2`，非零则按向量飞行并转向 → 支撑全向弹幕/防空/后跃齐射。

## set_deferred 护栏（按 9.2 教训）
- 9.3 新增群体技能(环身/全向/大招)更易在碰撞信号期触发死亡。已确认死亡/失效切碰撞仍走 `enemy._die`/`boss._die`/`projectile._fade` 的 `set_deferred`，**未新增任何直接 `monitoring=`/`collision_layer=` 写法**。
- **验证**：`SHOT_MOTION=1` 跑 **武器(sword/hammer/cannon) × 技能(ground/upper/dash/burst/ult) 共15组**(打 beast)，`Function blocked during in/out signal` + 脚本错误 **全部=0**；普通启动150帧无错。

## 复验建议（QA）
- 用 `SHOT_WEAPON=<sword/hammer/cannon> SHOT_SKILL=<ground/upper/dash/burst/ult>` 连拍对比三武器同技能的形态差异（harness 已支持切武器+burst）。
- 重点看：蒸汽炮 burst 的 360°弹幕、上挑的向上弹、突进的后跃；重锤 ult 的大爆炸/大半径。
- 手感项（霸体、后跃距离、挑飞高度）建议人工试玩；被动「机械超频」加点后技力条上限/回复变化。
