# 9.4 验收报告 — 华丽特效 + 数值平衡（单线程自我QA）

> 开发线程出具 · 日期 2026-06-27 · 按 `docs/qa/QA_PLAYBOOK.md` 闭环。
> 阶段9 收尾：每招特效增强 + 技力/怒气/伤害数值基线。

## 实现内容
**华丽特效（可画面验证）**
- 新增 `Fx.cast_ring`（技能起手：核闪 + 扩散环 + 火花）→ 每次放主动技/大招都有"放招那一下"的仪式感。
- 新增 `Fx.speed_lines`（横向能量速度线）→ 突进斩(铁剑/重锤冲撞)强调位移冲击。
- `_aoe_hit` 每次命中加 `Fx.hit_ring`（武器色冲击环）→ 群体命中更有打击感。
- 大招额外双 `cast_ring`（武器色 + 白）叠加屏闪/顿帧/震屏 → 终结技更隆重。

**数值平衡（首版基线，详见设计稿第四节）**
- 技力 100 / 回复 7 per s / 命中 +8；耗蓝 15/16/20/30；冷却 0.55/0.9/1.0/1.3。
- 怒气 命中+7 / 受击+12 / 放技+4；满 100 大招。
- **修了一处平衡不一致**：机械超频「过载输出 +15%」原先只作用于 upper/dash/burst/ult/dive（走 `_skill_dmg`），**地面波(`_heavy_*`)漏了** → 现加 `_oc()` 统一作用，全主动技一致受益。

## 诚实边界（待人工试玩）
- **数值平衡本身测不出**：伤害是否过强、技力是否够用、怒气攒速、冷却手感 → 全部需**人工试玩**逐档调。本阶段只给"能跑且不报错、特效到位"的首版基线。
- 特效"华丽度"是主观项：连拍证明了特效**触发且渲染**，但"够不够炫"需人眼+试玩判断。
- 上挑挑飞接空连、突进/冲撞位移手感同样待试玩。
