# 暗影机械城 · Shadow Mech City

> 2D 横版 **银河恶魔城（Metroidvania）动作游戏** — 暗黑机械 / 赛博蒸汽朋克风。
> 引擎：Godot 4.7 · 全代码程序化构建 · 开发中（Demo / 垂直切片）。

机械文明被「虚空浸蚀」吞没，暗影能量将机器化为怪物。你是唯一尚存意识的**机械猎魂者**，
深入这座被黑暗侵蚀的机械巨城，寻找重启文明的核心——**光核（Lumina Core）**。

---

## 🎮 操作

| 操作 | 按键 |
|------|------|
| 移动 | `A` / `D` |
| 跳跃（二段跳 / 蹬墙跳） | `空格` |
| 冲刺（穿越青色能量门） | `Shift` / `L` |
| 普通攻击（三连击 · `↑+J`上劈 · 空中`S+J`下砸） | `J` |
| 重攻击（各武器不同：剑气波 / 震地 / 远程炮） | `K` |
| 切换武器（铁剑 / 重锤 / 蒸汽炮） | `Q` |
| 投掷炸弹（落地弹跳引爆 · 炸开可破坏墙） | `F` |
| 技能树加点 | `T` |
| 背包 / 装备 | `U` / `I` |
| 世界地图 | `M` |
| 暂停 / 设置 | `Esc` |
| 重新开始 | `R` |

**能力（探索中获取）**：冲刺(初始) · 炸弹 · 攀墙(贴墙按↑/↓爬) · 滑翔(下落按住跳键)。
新能力可解锁原本到不了的区域与隐藏宝藏。

## ✨ 已实现

- **战斗**：3 把武器 + 各异重攻击、连击、暴击/吸血、命中顿帧/屏震/命中环、真实剑士攻击动画
- **成长**：技能树加点、经验/金币、装备背包（6 槽位 / 4 稀有度 / 强化 / 词条 / 掉落）
- **银河城世界**：8+ 互联房间、四向门、钥匙门、能力门（冲刺/炸弹）、存档塔、世界地图(M)
- **敌人**：9+ 种（行走/飞行/冲锋/重装/远程法师），带起手预警攻击
- **Boss**：2 个多阶段 Boss（蒸汽机甲·泰坦 / 岩核机甲巨兽，会召唤），独立血条、封门、击败奖励
- **外壳**：开始菜单 / 暂停 / 设置（音量·全屏）/ 存读档 / 键盘导航

完整规划见 [`docs/DEV_PLAN.md`](docs/DEV_PLAN.md)（约 35–40 小批次到商业成品）。
设计文档：[`docs/GAME_DESIGN.md`](docs/GAME_DESIGN.md) · 权威 GDD：`docs/design/暗影机械城_GDD(2).md`。

`E:\Godot\release\暗影机械城\`，双击 `ShadowMechCity.exe` 即玩（exe + 同名 .pck 需放一起）。

## 🗂️ 结构

```
scripts/   全部游戏逻辑(.gd)：player / enemy / boss / weapons / items_data /
           skill_panel / inventory_panel / map_panel / rooms / main / game(autoload) ...
assets/    程序生成 + 免费素材(瓦片/角色/敌人/特效/音频/背景)
docs/      设计文档(GDD/路线图/技能/地图/背包/特色系统/地图指引)
main.tscn  游戏主场景(空壳, 逻辑全在 main.gd)   title.tscn  开始菜单
```
